# Spy bot: телеграм-бот ведущий для игры в шпиона

С помощью бота можно сыграть компанией друзей в игру "Шпион". Доступно 2 режима игры: персонажи Dota 2 и карты Clash Royale. Бот выступает ведущим: сам хранит состояние игры и управляет процессом. В данный момент реализован функционал "один бот – одна игровая комната".

## Возможности

- Создание одной игровой комнаты
- Подключение нескольких игроков
- Минимум 3 игрока для старта
- Два режима игры: Dota 2, Clash Royale
- Автоматический выбор шпиона и персонажа
- Полный сброс состояния после завершения игры

## Использование:
- Вставьте токен своего бота в main.go
- Каждый игрок запускает бота (/start)
- Один из игроков создает игру
- Остальные игроки нажимают кнопку подключения к игре
- Каждый игрок может начать игру
- Каждый игрок может завершить игру, чтобы выбрать другой режим или пригласить еще друзей
